环境搭建

创建虚拟环境并安装依赖

  1. 安装虚拟环境: pip install virtualenv
  2. 安装虚拟环境管理包：pip install virtualenvwrapper-win
  3. 将虚拟环境管理工具添加到环境变量：变量名：WORKON_HOME变量值为存储虚拟环境的盘符路径
  4. 安装django框架与相关库依赖

创建项目及应用

  1. 可通过pycharm创建项目，也可通过命令行创建项目：django-admin startproject ProjectName
  2. 创建应用统一放在根目录下的apps文件夹下，让django读取应用将apps加入环境变量
  3. 进入到apps文件夹下：cd apps 然后 python manage.py startapp AppName
  4. 将应用注册到settings中的INSTALLED_APPS中

创建并配置数据库

  1. 创建一个本地用户以及本地数据库
  2. 赋予这个库的所有权限给这个用户
